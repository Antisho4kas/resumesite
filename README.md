# Сайт-резюме

Профессиональный сайт-резюме в бело-голубых тонах с современным дизайном и адаптивной версткой.

## 🎨 Особенности дизайна

- **Цветовая схема**: Белый и голубой тона
- **Современный дизайн**: Минималистичный и профессиональный вид
- **Адаптивность**: Отлично выглядит на всех устройствах
- **Анимации**: Плавные переходы и микро-взаимодействия

## 📁 Структура проекта

```
resume-website/
├── index.html          # Главная страница с мотивирующей надписью
├── links.html          # Страница с ссылками на соцсети и профили
├── files.html          # Страница с файлами для скачивания
├── contacts.html       # Страница контактов и форма связи
├── styles.css          # Стили сайта
├── script.js           # JavaScript функциональность
└── README.md           # Этот файл
```

## 🚀 Как использовать

### 1. Открытие сайта

Просто откройте файл `index.html` в вашем браузере:

```bash
# В Windows
start index.html

# В macOS
open index.html

# В Linux
xdg-open index.html
```

### 2. Персонализация

#### Изменение контактной информации

Откройте любой HTML файл и найдите секцию с контактами. Замените:

- `email@example.com` → ваш email
- `+7 (XXX) XXX-XX-XX` → ваш телефон
- `@yourusername` → ваши никнеймы в соцсетях
- Ссылки на профили → ваши реальные ссылки

#### Настройка ссылок (links.html)

Замените плейсхолдеры ссылок:

```html
<a href="https://linkedin.com/in/yourprofile" target="_blank">
<a href="https://github.com/yourusername" target="_blank">
<a href="https://t.me/yourusername" target="_blank">
```

#### Добавление файлов (files.html)

Для добавления реальных файлов:

1. Создайте папку `files` в корне проекта
2. Поместите туда ваши файлы
3. Обновите ссылки в `files.html`:

```html
<a href="files/your-resume.pdf" download>
```

#### Настройка формы контактов

Форма в `contacts.html` готова к интеграции с бэкендом. Для полной функциональности:

1. Добавьте обработчик формы в `script.js`
2. Интегрируйте с сервисом отправки писем (Formspree, Netlify Forms и т.д.)

## 🚀 Развертывание

### GitHub Pages

1. Загрузите все файлы в репозиторий GitHub
2. В настройках репозитория включите GitHub Pages
3. Выберите ветку `main` и папку `/root`

### Netlify

1. Загрузите файлы на Netlify
2. Сайт будет автоматически развернут

### Локальный сервер

Для локальной разработки с сервером:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server
```
